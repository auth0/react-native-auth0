    Interface IMfaClient

    Defines the contract for MFA operations using the Flexible Factors Grant.

    An MFA client is scoped to a single MFA flow, initialized with an mfaToken from an MFA_REQUIRED error. It provides methods to list authenticators, enroll new factors, challenge existing factors, and verify MFA codes.
