    Interface IPasswordlessClient

    Client for the passwordless OTP flow on database connections.

    This is a two-step, challenge-response flow: issue a challenge to an email or phone number (which delivers a one-time code and returns an opaque auth_session), then complete the login by verifying the code.

    Remarks

    Native only (iOS, Android). Not supported on web.
