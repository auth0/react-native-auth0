    Interface PasskeySignupChallengeParameters

    Parameters for requesting a passkey signup challenge from Auth0.

    The challenge response contains WebAuthn creation options that should be passed to the platform's credential manager (CredentialManager on Android, ASAuthorizationController on iOS) to create a new passkey credential.

    See

    https://auth0.com/docs/authenticate/database-connections/passkeys
